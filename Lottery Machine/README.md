# Lottery Machine
 lottery machine, 로또 추첨기
 <br>
 사용자가 번호를 선택할 수 있고, 번호생성을 통해 랜덤번호 생성가능한 어플리케이션
 <br>
 
 V 1.0.0
 - NumberPicker를 활용하여 번호선택/번호추가
 - 초기화버튼을 통해 번호 초기화 가능
 - <번호 생성> 버튼을 클릭하여 번호 추출 가능
